## ☁️ Cloud Service Models (IaaS, PaaS, SaaS)
## 🔹 IaaS (Infrastructure as a Service)
- Provides only infrastructure (servers, storage, networking)
You are responsible for managing OS and software
- ✅ AWS Examples:
- EC2
- EBS
- VPC
#### 📌 Short Definition:
"In IaaS, the user gets infrastructure and manages everything on their own."
## 🔹 PaaS (Platform as a Service)
- AWS provides a ready-to-use platform
You only focus on application development
- ✅ AWS Examples:
- Elastic Beanstalk
- RDS
#### 📌 Short Definition:
- In PaaS, the developer focuses only on the application while AWS manages the rest."
## 🔹 SaaS (Software as a Service)
- Ready-made software is provided
- The user simply uses it
- ✅ Examples:
- Gmail
- Google Drive
AWS: WorkDocs, Chime
### 📌 Short Definition:
- In SaaS, users get ready-to-use software that can be used directly."
